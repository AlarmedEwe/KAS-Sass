Instale o

Para utilizar o Framework KAS é preciso ter instalado na sua máquina o , que pode ser adquirido com a ajuda do Node.

$ npm install -g sass

Compile seu código

Com o instalado na máquina, basta abrir o Prompt de Comando, acessar a pasta do seu projeto e executar o seguinte código:

(Não se esqueça de substituir "_sass" pelo diretório de origem e "_css" pelo diretório de destino)

$ sass --watch _sass/KAS.scss _css/KAS.css
